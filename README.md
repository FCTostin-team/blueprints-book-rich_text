A comprehensive web tool designed for decoding, editing, and re-encoding Factorio blueprint strings. It allows users to view the internal JSON structure of blueprints and blueprint books, modify metadata, and manage blueprint icons directly in the browser.

## Features

* **Blueprint Decoding**: Convert Factorio blueprint strings into readable JSON format.
* **JSON Editor**: Integrated CodeMirror editor for direct manipulation of blueprint data.
* **Blueprint Books**: Specialized tools for editing blueprint book metadata, including labels, descriptions, and active icons.
* **Search & Replace**: Built-in functionality to find and replace values within the blueprint JSON.
* **History**: Automatically saves decoded blueprints locally for quick access.

## Installation and Usage

1.  Clone the repository or download the source files.
2.  Open `index.html` in any modern web browser.
3.  Paste a blueprint string to begin editing.
